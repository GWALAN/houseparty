House Party – Privacy Policy

Last updated: December 2025

House Party Score Tracker (“the App”) is developed by Elandre (“Developer”). This Privacy Policy explains how the App handles user data.

1. Information We Collect

The App does not collect personal information such as names, phone numbers, or contacts.

The App may store:

  • User-created scores
  • Player names entered manually
  • Optional images uploaded inside the app
  • Basic analytics data (device type, crash logs)

All data is stored securely using Supabase, a GDPR-compliant cloud provider.

2. How We Use Data

Data is used only to:

  • Save player scores
  • Display leaderboard information
  • Improve app performance
  • Debug crashes and errors

We do not sell, share, or use data for advertising.

3. Data Security

All communication with the database is encrypted using HTTPS.
Supabase encrypts data at rest.

4. Children’s Privacy

The App does not target children under 13. No sensitive child data is collected.

5. Third-Party Services

The App uses:

  • Supabase (data storage)
  • Expo and React Native libraries

These services may collect diagnostic data.

6. Data Retention

Game data is retained for as long as the user keeps the account or until it is deleted manually.
